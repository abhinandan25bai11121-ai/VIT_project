# VIT_project  
A Python data-analysis project on top 100 movies.

## About  
This project explores a dataset of the top 100 movies (in CSV format) and provides a Python script (#MOVIE NAME.py) to perform analysis / extraction of movie-names / other features.  
It is created as part of a project for VIT (presumed as your institution) to demonstrate data handling, visualization & basic analytics.

## Contents  
- top_100_movies.csv - the dataset containing movie records.  
- #MOVIE NAME.py - Python script to work with the dataset.  
- README.md - this document.  
- Project_Report.pdf (to be added) - detailed report of methodology, findings & conclusions.
